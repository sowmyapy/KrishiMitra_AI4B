# Make Real Voice Call - Complete Guide

## Overview

This guide will help you make an actual voice call to your phone number (+918095666788) with a real agricultural advisory in Hindi.

## Prerequisites

### 1. Twilio Account Setup

You need a Twilio account with:
- Account SID
- Auth Token  
- A Twilio phone number (with voice capability)

**Get Twilio Account**:
1. Go to https://www.twilio.com/try-twilio
2. Sign up for free trial (includes $15 credit)
3. Verify your phone number (+918095666788)
4. Get a Twilio phone number (choose India +91 number if possible)

**Find Your Credentials**:
1. Go to Twilio Console: https://console.twilio.com/
2. Copy Account SID
3. Copy Auth Token
4. Copy your Twilio Phone Number

### 2. Update .env File

Add to your `.env` file:
```env
TWILIO_ACCOUNT_SID=your_account_sid_here
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_PHONE_NUMBER=+1234567890  # Your Twilio number
```

### 3. Public Webhook URL (Required)

Twilio needs a public URL to send call events. You have 3 options:

#### Option A: ngrok (Recommended for Testing)

1. **Install ngrok**:
   - Download from https://ngrok.com/download
   - Extract and add to PATH

2. **Start your server**:
   ```powershell
   .\venv\Scripts\Activate.ps1
   uvicorn src.main:app --reload
   ```

3. **Start ngrok tunnel** (in new terminal):
   ```powershell
   ngrok http 8000
   ```

4. **Copy the HTTPS URL**:
   ```
   Forwarding  https://abc123.ngrok.io -> http://localhost:8000
   ```
   
   Your webhook URL will be: `https://abc123.ngrok.io/voice/advisory`

#### Option B: Twilio TwiML Bins (Simplest)

1. Go to Twilio Console > TwiML Bins
2. Create new bin
3. Paste the TwiML (generated by script)
4. Save and use the bin URL

#### Option C: Production Deployment

Deploy your app to AWS/Heroku and use that URL.

## Troubleshooting

### Error: Twilio credentials not found

**Solution**: Add credentials to `.env` file:
```env
TWILIO_ACCOUNT_SID=ACxxxxx
TWILIO_AUTH_TOKEN=your_token
TWILIO_PHONE_NUMBER=+1234567890
```

### Error: Phone number not verified

**Solution**: 
1. Go to Twilio Console > Phone Numbers > Verified Caller IDs
2. Add +918095666788
3. Verify via SMS code

### Error: Webhook URL not accessible

**Solution**:
- Make sure ngrok is running
- Use HTTPS URL (not HTTP)
- Check firewall settings
- Test URL in browser first

### Error: Call fails immediately

**Possible causes**:
1. Insufficient Twilio credits
2. Phone number not verified (trial account)
3. Webhook URL not responding
4. Invalid TwiML

**Check**:
- Twilio Console > Logs > Calls
- See detailed error messages

### No audio / Silent call

**Solution**:
- Check TwiML is valid
- Verify webhook returns proper TwiML
- Check Twilio debugger logs
